Submission

If you want to help improving iakesi's database, you can help submit with any information you have.

What to Submit?

The information you can submit are as follow:

  • Better location description
  • Better location photo
  • Better screenshot from anime
  • Whole new entry

If you're submitting a whole new entry, please make sure to provide these:

  • Location name
  • Location address (country, prefecture/province/state, city/town)
  • Location description
  • Location image
  • Location latitude and longitude
  • The series the location appears in
  • The location screenshot from the said series

Where to Submit?

As there is no proper functionality to make a submission, as of now, please submit by sending email to: iakesi@protonmail.com

Credits

If approved, the description, photo, and screenshot you submitted will be properly credited.
