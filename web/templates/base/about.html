About iakesi

iakesi exists with the purpose to map real life anime locations to assist your next anime pilgrimage. The goal is to have every location ever shown in anime properly is mapped and archived in iakesi.

If you would like to help improve the database, please head to Submission to see how you can help.

iakesi's source code is available on GitHub, if you're interested.
